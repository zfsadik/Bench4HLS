Create a 16-bit register with byte-enable write control. The byte-enable inputs (byteena) control whether each byte of the register should be written to on that cycle. byteena[1] controls the upper byte d[15:8], while byteena[0] controls the lower byte d[7:0]. resetn is a synchronous, active-low reset. The design has a 16-bit data input (d) and a 16-bit output (q).

The top-level function should have the following prototype:
void TopModule(bool resetn, ap_uint<2> byteena, ap_uint<16> d, ap_uint<16> &q)
