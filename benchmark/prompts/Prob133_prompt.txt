Implement the six‑state FSM below with synchronous active‑high reset. Input is w and output is z.

    A(z=0): w=1--> B, w=0--> A
    B(z=0): w=1--> C, w=0--> D
    C(z=0): w=1--> E, w=0--> D
    D(z=0): w=1--> F, w=0--> A
    E(z=1): w=1--> E, w=0--> D
    F(z=1): w=1--> C, w=0--> D

Reset synchronously returns the FSM to state A on the rising clock edge.

The top-level function should have the following prototype:
void TopModule(bool reset, bool w, bool &z)
